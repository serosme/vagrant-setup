## 命令

Global Status

```bash
vagrant global-status
```

Up

```bash
vagrant up
```

Halt

```bash
vagrant halt
```

Destroy

```bash
vagrant destroy -f
```

Snapshot Save

```bash
vagrant snapshot save [vm-name] NAME
```

Snapshot Restore

```bash
vagrant snapshot restore [vm-name] NAME
```
